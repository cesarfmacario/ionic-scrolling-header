# ionic-scrolling-header
An angular directive to make the header hide with scrolling

**version 0.1.0**

## Usage

* add ` @ViewChild(Content) content: Content;` in your component

* and add 


`<ion-header [scrollingHeader]='content'>
    <ion-navbar>
        <ion-title>
            Contact
        </ion-title>
    </ion-navbar>
</ion-header>`
* :) good to go
